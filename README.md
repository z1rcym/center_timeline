# CenterTimeline



## インストール方法

`.jsx` ファイルを **ScriptUI Panels** フォルダに入れ、After Effects の「ウィンドウ」メニューから実行してください。



## 機能

選択しているレイヤーのタイムラインを中央に移動させることができます。

複数のレイヤーを選択している場合は、一番最初に選択したレイヤーが対象となります。
